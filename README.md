# GoProRemote for Pebble

Unofficial GoPro Remote for Pebble watches

##Supported watches:

* Pebble OG/Steel
* Pebble Time/Time Steel
* Pebble Time Round
* Pebble 2

##Supported cameras:

* GoPro HERO3/3+
* GoPro HERO4 (incl. Session)
* GoPro HERO5 (incl. Session)
* GoPro HERO+/HERO+ LCD
